# Catch Jiu Jitsu Portal

Portal app with React frontend and Laravel backend.

## Run Locally

**Prerequisites:** Node.js, PHP 8.2+, Composer

1. Install frontend dependencies: `npm install`
2. Set `GEMINI_API_KEY` in `.env.local` (optional, for AI features)
3. Run the app: `npm run dev`

For full setup with Laravel backend, see [DEPLOY.md](DEPLOY.md) and [COOLIFY.md](COOLIFY.md).
